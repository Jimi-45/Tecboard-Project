🚀 Tecboard Project
Este é o meu primeiro projeto de desenvolvimento web, criado durante os cursos da plataforma Alura. O site é uma landing page focada em monitoramento de aplicações em tempo real, com um design moderno e responsivo.
📋 Sobre o Projeto
O Tecboard foi desenvolvido para colocar em prática conceitos fundamentais de HTML e CSS. O objetivo é apresentar uma solução de monitoramento com alertas inteligentes, garantindo que aplicações fiquem sempre online.
🛠️ Tecnologias Utilizadas

    HTML5: Estruturação de conteúdo.
    CSS3: Estilização, uso de Flexbox e fontes personalizadas.
    Google Fonts: Utilização da família de fontes Inter.

🎨 O que eu aprendi
Neste projeto, pude consolidar conhecimentos em:

    Estrutura semântica de tags HTML.
    Alinhamento de elementos com Flexbox.
    Criação de botões com efeitos de hover.
    Importação de imagens e ícones externos.
